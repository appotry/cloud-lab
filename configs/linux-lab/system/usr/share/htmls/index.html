Linux Lab 开源项目

Linux Lab 致力于提供一个『开箱即用』的 Linux 内核与嵌入式 Linux 系统开发环境。

  • https://github.com/tinyclub/linux-lab

主要特性

  1. 支持 3 大操作系统（Windows, MacOS, Linux），可以轻松在这三大操作系统下使用。
  2. 支持 7+ 大处理器架构（X86, ARM, MIPS, PPC, RISC-V, LOONGSON, CSKY），其中 LOONGSON 和 CSKY 为国产处理器。
  3. 支持 25+ 款开发板（i386/pc, x86_64/pc, arm/versatilepb, arm/vexpress-a9, arm/mcimx6ul-evk, ppc/g3beige, mips/malta, aarch64/virt, aarch64/raspi3, riscv32/virt, riscv64/virt, csky/virt, mipsel/ls1b, mipsel/ls232, mips64el/ls2k, mips64el/ls3a7a, arm/ebf-imx6ull）。
  4. 支持 5 种登陆方式（bash, ssh, vnc, webssh, webvnc），可以本地访问，也可以远程访问。
  5. 集成了 5 大组件（Qemu, U-boot, Buildroot, Linux, Toolchain），都有预编译版本。
  6. 内置了 5 大平台，32 位和 64 位共 10 个 Hello World 汇编语言例程，见 examples/assembly。
  7. 可以学习处理器指令集、Qemu、Shell、汇编、C、Linux 内核、嵌入式 Linux。
  8. 支持 Debugging 和 Testing。
  9. host & guest 双侧免 root 使用。

用法一：独立开机启动

  • 找一台 X86_64 的 PC 或 笔记本，关掉电源
  • 找到 Type-A/C USB 口并插入 Linux Lab Disk
    • 如主机不支持 Type-A/C 口，请使用相应转接口
  • 按住 F12 同时按下电源键直到看到引导设备列表
    • 如 F12 不工作，请根据自己 PC 或笔记本型号去网络检索
    • 采用 X86_64 处理器的 macBook 也支持，可用 Option 按键
    • 如果启动时遇到 安全/Security 相关错误提示，请通过笔记本的 BIOS 禁用/Disable 安全引导/Security Boot，具体方法因笔记本型号而异，可通过网络检索
  • 选中新增的磁盘设备名（因 Linux Lab Disk 型号不同会有差异）并启动
  • 启动后在桌面会看到 Linux Lab 图标，选中并点击，稍等片刻即可看到浏览器中的 Linux Lab 环境

用法二：在运行的系统上并行启动

  • 找一台已验证的系统：Windows 7/8/10/11, Ubuntu 18.04/20.04/21.04/22.04, Deepin 20.2+, Mint, Manjaro, Kali, Fedora, 更多系统正在陆续开发与验证中
  • 插入 Linux Lab Disk
    • 一般会自动挂载 linux-lab-disk 分区，Linux 会挂载在 `/media/$USER/linux-lab-disk`，Windows 下则可以在文件夹浏览器的左侧查看到
    • 找到对应版本的 vmboot zip 包，复制到主系统的桌面并解压
    • Windows 下直接点击运行 install.bat，Linux 则可以启动一个终端来运行 install.sh 脚本
  • 如果本机已经安装 Virtualbox 或 VMware，则会自动启动 Linux Lab Disk
  • 如果未安装过 Virtualbox 或 VMware，Windows 下可以直接安装 linux-lab-disk 分区中提供的 Virtualbox 安装包，Linux 下则需要根据发行版安装相应版本，也可以直接去 Virtualbox 官网 下载；在开启 Windows WSL2 的情况下，必须使用 VMware Player 才能正常启动，请自行到 VMware 官网 下载 17 以上版本
  • 安装完 vmboot 过后，插入 Linux Lab Disk 就会自动启动
  • 安全起见，在拔掉 Linux Lab Disk 之前，请务必做到：
    • 在 Virtualbox 运行的 Linux Lab Disk 系统内部，正常关闭该系统
    • 确保 Linux Lab Disk 所有分区数据都已经落盘
    • 安全卸载 linux-lab-disk 等分区（64G或以上容量在 Linux 下会会自动挂载 misc 分区）
    • 为了防止使用过程中误拔，可以使用贴纸等把 Linux Lab Disk 固定到电脑上

特性

  • 主系统已适配 Ubuntu 18.04/20.04/21.04/22.04, Deepin 20.2+, Mint, Fedora, Manjaro, Kali 等，更多主系统正在适配中
  • 内置 Linux Lab, Linux 代码仓库等
  • 免安装，零门槛学 Linux
  • 开机上电就用 + 在 Windows、Linux 下当双系统用
  • 内置 Linux 0.11, Linux, CS630 汇编等实验
  • 采用 USB 3.x, M.2 NVME 或 SATA 等接口, 选用高速或固态等控制器
  • 有 64G/128G/256G/512G/1T/2T 等可选
  • 支持透明倍容，实际可用容量高出10%～50%
  • 支持出厂恢复，再也不用担心把系统搞死了！而且很方便通过 timeshift 按需备份还原！
  • 支持内存编译，加速编译，提升磁盘寿命
  • 还可以当急救盘、资料转移盘（预留有 NTFS 分区）使用
